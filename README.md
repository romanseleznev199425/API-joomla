# API-joomla
API приложение для joomla 
Приложение API joomla для создания материалов из json данных. 
Для создания материалов надо указать url и API токен пользователя сайта. После загрузить файл json для создания материалов. 
Формат данных JSON:
{
    "articles": [
        {
            "title": "Заголовок",
            "alias": "alias",
            "introtext": "<p>Привью текст</p>",
            "fulltext": "<p>Полный текст</p>",
            "image_intro": "Привью изображение",
            "image_intro_alt": "Описание привью изображения",
            "metadesc": "",
            "metakey": "",
            "created": "24.11.2025, 12:12:12",
            "publish_up": "24.11.2025, 12:12:12",
            "state": "Статус"
        }
    ]
}
